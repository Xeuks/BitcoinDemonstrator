Was ist eine Blockchain?

Wie der Name schon aussagt, handelt es sich bei einer Blockchain, um eine Kette von Blöcken. Dabei wird bei Bitcoin in den Blöcke eine Sammlung von Transaktionen abgespeichert. Ein Block hat immer eine Referenz auf den vorhergehenden Block. Folgende Visualisierung zeigt eine Blockchain mit 4 Blöcken auf.

Die Verkettung der Blöcke wird gebildet indem, ein Block auf den vorhergehenden Block anhand seines Hashwertes verweist. Es stellt sich die Frage was ist ein Hashwert? Der Hashwert wird anhand einer Hashfunktion gebildet. Bei Bitcoin verwendet man die SHA256 Hashfunktion. Diese Funktion bildet aus einem Input eine 64-stellige Zeichenfolge, der sogenannte Hashwert bzw. Hash. Eine spezielle Eigenschaft des Hashwertes ist, dass anhand des Hashwertes nicht ermittelt werden kann, was der Input Text ist.
